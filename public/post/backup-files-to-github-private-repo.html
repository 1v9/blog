Access token

访问 Personal Access Tokens 点击 Generate new token 按钮。

关于安全性

  • 不要推送任何可能泄露账户与个人隐私等危险信息的文件。
  • 妥善保管 ssh key 与 access token，如果遗失或者仅仅尝鲜，请自行通过 Github 网站取消它们的授权。
  • 正常来说，启用 Github 账户两步验证、多注意邮箱通知、不要在不熟悉的设备上操作，没有太大问题。
